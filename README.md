# API Test for Authentication

Selamat datang di proyek **API Test for Auth**! 🎉 Proyek ini dirancang untuk memastikan bahwa sistem autentikasi berjalan dengan lancar melalui serangkaian pengujian otomatis.

## 🚀 Fitur
- 🔐 Pengujian API autentikasi.
- 📦 Manajemen dependensi dengan **npm**.
- ✅ Pengujian otomatis menggunakan perintah sederhana.

---

## 🛠️ Instalasi

Ikuti langkah-langkah di bawah ini untuk memulai:

### 1. Clone Repository
```bash
$ git clone <URL-REPOSITORY>
$ cd api-test-auth
```

### 2. Instal Dependensi
Jalankan perintah berikut untuk menginstal semua dependensi yang diperlukan:
```bash
$ npm install
```

---

## 🧪 Menjalankan Pengujian

Untuk menjalankan pengujian dan memastikan API Anda berfungsi dengan benar, gunakan perintah berikut:
```bash
$ npm test
```

Setelah perintah dijalankan, Anda akan melihat hasil pengujian langsung di terminal.

---

## 🌟 Tips
- Pastikan **Node.js** dan **npm** terinstal pada mesin Anda.
- Selalu jalankan `npm install` setelah memperbarui kode untuk memastikan dependensi tetap sinkron.
